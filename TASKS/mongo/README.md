# Mongo Task

Simple CRUD API using Express and Mongoose.

## Endpoints

- `GET /api/users`
- `GET /api/users/:id`
- `POST /api/users`
- `PUT /api/users/:id`
- `DELETE /api/users/:id`

## Notes

- Copy `.env.example` to `.env`
- Make sure MongoDB is running locally
- Run `npm install`
- Run `npm run dev`
